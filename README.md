# Developer Portfolio (Next.js 16 App Router)

Minimal, dark-mode-first developer portfolio scaffold using Next.js 16 (App Router), Tailwind CSS, and Framer Motion. Project data is stored locally in `data/projects.js` (no database).

Quick start

```bash
cd ~/CV
npm install
npm run dev
```

Open `http://localhost:3000`.

Notes

- Edit `data/projects.js` to add your projects.
- Components live in `components/`.
